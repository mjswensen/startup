Layout

Grid System

Layouts are controlled with column and row classes with appropriate numbers appended. The grid is based on a 12-column layout, and is percent-based. Wrap your layout in a .grid-container.

Columns

.col-3

.col-7

.col-2

Rows

.row-1

.row-2

.row-3

.row-4

.row-5

.row-6

Note: row height for all layout widths is based on the column width for a grid wrapped in a 960-pixel-wide container.

Multiple Rows

To include multiple rows within a single .grid-container, include the last-col class on the last elements of a row. This isn't necessary if the .grid-container wraps only one row in your layout.

.col-4.row-1

.col-4.row-1

.col-4.last-col.row-1

.col-6.row-1

.col-6.row-1

Containers

There are a few different container widths to choose from, using the following classes: .fluid (viewport width), .fixed (standard 960-pixel-wide horizontally centered container), .fixed-thin (500-pixel-wide horizontally centered container), and .fixed-wide (1080-pixel-wide horizontally centered container).
